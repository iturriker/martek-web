# Imágenes Necesarias para Martek Web

Este documento lista todas las imágenes que deben estar en la carpeta `assets/images/` para que la web funcione correctamente.

## Imágenes Actuales Requeridas

### Logo y Branding
- `martek-logo.png` - Logo principal de Martek (debe existir)

### Página Principal
- `hero.jpg` - Imagen de fondo del hero (mangueras industriales, almacén, etc.)

### Productos
- `mangueras-industriales.jpg` - Foto de mangueras industriales variadas
- `acoplamientos.jpg` - Foto de diferentes tipos de acoplamientos
- `columnas-riego.png` - Foto de columnas de riego DN-80
- `flexibles-aire.jpg` - Foto de flexibles y mangueras neumáticas

## Favicon

Se necesita crear un favicon. Opciones:
1. **Opción recomendada**: Crear un favicon.ico de 32x32px con el logo simplificado de Martek
2. También crear versiones PNG para diferentes dispositivos:
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `apple-touch-icon.png` (180x180px)

Los favicons deben colocarse en la raíz del proyecto.

## Recomendaciones

### Optimización de Imágenes
Todas las imágenes deben estar optimizadas para web:
- Formato JPG para fotografías (calidad 80-85%)
- Formato PNG para logos y gráficos con transparencia
- Tamaño recomendado para productos: máximo 1200px de ancho
- Tamaño recomendado para hero: 1920px de ancho

### Herramientas de Optimización
- TinyPNG.com - Para comprimir sin pérdida de calidad
- Squoosh.app - Herramienta de Google para optimización
- ImageOptim - Para Mac

### Imágenes Adicionales Opcionales
Si se desean mejorar las páginas de productos, podrían añadirse:
- Imágenes específicas de cada tipo de manguera
- Fotos de instalaciones y proyectos realizados
- Imágenes del equipo o almacén
- Certificaciones y logos de fabricantes

## Próximos Pasos

1. Recopilar las imágenes existentes del negocio
2. Optimizarlas con las herramientas mencionadas
3. Renombrarlas según la nomenclatura indicada
4. Colocarlas en las carpetas correspondientes
5. Crear los favicons a partir del logo

Si no tienes algunas imágenes, puedo sugerir bancos de imágenes gratuitos o de pago donde encontrar fotografías profesionales de productos industriales.
